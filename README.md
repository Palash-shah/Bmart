# Bmart - E-Commerce Website

## Summary:
Hi everyone! ❤️

This is a very simple e-commerce website which was made by me as a final year project of my diploma engineering dedicated to the sale of vegitable and fruits.

## Platform Used
### Front-End
* HTML5
* CSS3
* Bootstrap
* JavaScript

### Back-End
* PHP
* MySQL

## Features
### Public User
* Search Product
* View Product
* Create User Account

### Signin User
* Search Product
* View Product
* Add To Cart
* Buy Now
* Edit Profile
* Can View Previous Order

### Admin
* Add New Product
* Update Product
* Delete Product
* Contact us Request

## Conclusion
There are also many more feature which are not in the list. Feel free to use this project
